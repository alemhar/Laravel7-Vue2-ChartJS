## Project Information/Prerequisite

- Laravel 7
- PHP 7.3
- MySQL
- VueJS
- npm 
- composer


## Installation

- Clone or Download source code to a directory. 
- On the Terminal, Change Directory to the directory.
- Run "composer update"
- Run "npm install"
- Run "php artisan serve"
